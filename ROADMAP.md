# Roadmap

## Planned Features

### Edit View Mode

Mark individual anime as hidden in the calendar without removing them from your list. Toggle into selection mode from a button, check/uncheck anime to hide them, then return to normal view. Hidden anime won't appear in the calendar until you un-hide them in edit mode.

### Mobile Responsive Improvements

- Single-column calendar layout on mobile (instead of full grid)
- Day-of-week shortcuts (Mon/Tue/Wed etc.) as sidebar buttons on left
- Tap a day button to scroll that day into view

### Vertical Calendar Layout

Alternative layout mode where days are rows (not columns), rendered as a scrollable list instead of a grid. Configurable in settings.

### Touch-Friendly Card Interactions

Replace hover-only information on anime cards with a touch-compatible mechanism. Options:

- Tap to expand card details
- Long-press to show info
- Slide gesture to reveal hidden content
- Persistent info bar at bottom of mobile screens

### Multilenguaje

Add i18n support for Spanish, English, French, and German. Configure via settings and persist language preference to localStorage.

### Keyboard Accessibility Review

Audit and test keyboard navigation across all features. Ensure tab order is logical, all interactive elements are reachable, and keyboard shortcuts are intuitive.

### Light Mode Review

Test and refine the light theme variant. Ensure contrast ratios meet WCAG standards and visual hierarchy is clear in light mode.
